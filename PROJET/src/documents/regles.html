Règles simplifiées Fanorona

Début de partie :

Un plateau est constitué de 45 intersections sur lesquelles sont répartis les pions de chaque joueur sauf sur l'intersection centrale.

Déroulement de partie :

Chaque joueur jouera l'un après l'autre : pour ce faire, il faut déplacer un pion sur une intersection adjacente libre connectée. Il mangera alors tous les pions ennemis successifs devant lui (percussion) ou derrière lui (aspiration) dans le sens de son déplacement. Si aucun mouvement offensif n'est possible, il peut jouer un coup non offensif.
Après ce déplacement, le joueur peut continuer à jouer tant qu'il peut faire d'autres déplacements offensifs avec ce même pion, dans une nouvelle direction et sans passer par un point précédemment visité.

But du jeu :

Un joueur sera déclaré vainqueur quand il aura mangé tous les pions de l'adversaire ou si son adversaire se retrouve dans l'incapacité de jouer le moindre coup.
Si aucune prise n'est faite pendant 40 tours, la partie sera déclarée nulle.
